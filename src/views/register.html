In the past, student organizations have found inconsistencies in the number of voters and the number of votes cast. This leads to the ballot being spoilt and another round of voting has to held. Furthermore, manually counting votes and following intricate rules can be tedious and lead to mistakes by election committees.

This website aims to solve such inconsistencies by automatically counting votes and enforcing strict voting rules while maintaining voter privacy and anonymity. We strive to make this website very straight forward and user friendly towards both the voters and the election administrators.

The website is open source and created for free by Waseem Ahmad and Andrew Capshaw. The source code can be viewed here.
